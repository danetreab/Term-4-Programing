Linotype
Line-castion allowed type to be selected, used, then recirculated into the machine automatically. This advance increase the speed of typesetting and printing dramatically
Photocomposition
Typefaces are stored on film then projected onto photo-sensitive paper. Lenes adjust the size of the type.
